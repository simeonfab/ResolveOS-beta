# Start Here

ResolveOS helps ChatGPT understand, organise, continue, and improve projects.

You do not need to choose roles, workflows, modes, or technical setup. Describe your project in plain English and ResolveOS should work out the right next step.

## What To Do First

Send ChatGPT one short message like:

```text
I want to use ResolveOS for this project. Here is what I have and what I want help with: ...
```

## What You Can Give It

You can provide:

- a new idea
- a GitHub repository
- a Notion workspace or page
- a Jira project or ticket
- existing docs
- a business process
- a partially complete project
- notes about what is blocked or confusing

## What Happens Next

ResolveOS should identify whether your project is new, existing, or already in progress.

Then it should ask only for missing essentials and recommend the smallest useful next step.

You do not need to understand ResolveOS internals before you begin.
